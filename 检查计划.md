# dgh-HealthCraft 检查计划

## 1. 配置项
- [ ] 胶囊两次服药间隔可配置（默认300秒）
- [ ] 胶囊服药后产生作用延迟可配置（默认120秒）
- [ ] 阻断剂有效期可配置
- [ ] 注射器耐久度可配置
- [ ] 可配置实体类型致病（尸毒感染）

## 2. 药品与注射相关
- [ ] 胶囊服药逻辑（间隔、延迟）
- [ ] 主手药针、副手注射液配合使用逻辑
- [ ] 注射液使用后减少耐久度
- [ ] 受污染药针有概率感染脓毒症、艾滋病
- [ ] 阻断剂注射液使用后变成受污染药针
- [ ] 利巴韦林使用后变成受污染药针

## 3. 疾病系统
- [ ] 艾滋病：受污染药针概率感染
- [ ] 艾滋病增加其他疾病概率
- [ ] 拉米夫定胶有10%概率治疗艾滋病
- [ ] 上呼吸道感染（轻/中/重型）
    - [ ] 雨天暴露行为概率感染
    - [ ] 轻型引发挖掘疲劳
    - [ ] 放任恶化为中/重型
    - [ ] 睡觉概率自愈（重型无法自愈）
    - [ ] 蛋白质过低/艾滋病增加概率
    - [ ] 右美沙芬治疗轻型/概率降级中型
    - [ ] 布洛芬暂时降级中/重型为轻型
    - [ ] 利巴韦林立即降级中/重型为轻型
- [ ] 创伤后应激障碍
    - [ ] 被攻击时对来源恐惧
    - [ ] 低血量时被攻击受虚弱等效果
    - [ ] 睡觉缓解
    - [ ] 镇静剂缓解
- [ ] 脓毒症：受污染药针概率感染
    - [ ] 行动时额外消耗糖类/油脂
    - [ ] 有概率中毒
    - [ ] 广谱抗生素可治疗
- [ ] 尸毒感染
    - [ ] 亡灵生物攻击概率感染
    - [ ] 感染后上呼吸道感染无法自愈
    - [ ] 睡觉无法自我调节
    - [ ] 随时间恶化，尸变后全身恶化，最终死亡
    - [ ] 靶向剂缓解尸变恶化
    - [ ] 阻断剂短时间免疫尸毒
